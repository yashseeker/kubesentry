# KubeSentry

A cloud-native Incident Management platform built with Spring Boot and Kubernetes.

## Features

- Incident REST APIs
- PostgreSQL database
- Dockerized application
- Kubernetes Deployment
- Kubernetes Secrets
- Persistent Storage
- Rolling Updates
- Self Healing
- Scaling

## Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Maven
- Docker
- Docker Hub
- Kubernetes

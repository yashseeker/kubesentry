# KubeSentry

A cloud-native Incident Management platform built with Spring Boot and Kubernetes.

- PostgreSQL database
- Dockerized application
- Kubernetes Deployment
- Kubernetes Secrets
- Persistent Storage
- Rolling Updates
- Self Healing
- Scaling

- Spring Boot
- PostgreSQL
- Maven
- Docker
- Docker Hub
- Kubernetes
